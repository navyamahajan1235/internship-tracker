# Internship Tracker

A full-stack web application built using Flask, SQLite, HTML, and CSS to track internship applications.

## Features

• User registration and login  
• Add internship applications  
• Edit applications  
• Delete applications  
• Clean and modern UI  
• Secure session-based authentication  

## Tech Stack

Frontend: HTML, CSS  
Backend: Python (Flask)  
Database: SQLite  
Version Control: Git & GitHub  

## How to run locally

1. Clone repository
git clone https://github.com/navyamahajan1235/internship-tracker.git

2. Go to folder
cd internship-tracker

3. Create virtual environment
python -m venv venv

4. Activate
source venv/bin/activate

5. Install requirements
pip install -r requirements.txt

6. Run app
python app.py
