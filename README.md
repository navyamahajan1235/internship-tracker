# Internship Tracker

A full-stack web application to track internship applications with search, filtering, editing, and dashboard analytics.

## Features

- User registration and login
- Add internship applications
- Edit and delete applications
- Search by company name
- Filter by status (Applied, Interview, Offer, Rejected)
- Dashboard with statistics cards
- Created date tracking
- Clean modern UI

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS

## How to run locally

1. Clone repository

git clone https://github.com/navyamahajan1235/internship-tracker.git

2. Go to folder

cd internship-tracker

3. Create virtual environment

python -m venv venv

4. Activate environment

Mac/Linux:
source venv/bin/activate

Windows:
venv\Scripts\activate

5. Install requirements

pip install -r requirements.txt

6. Run app

python app.py

7. Open browser

http://127.0.0.1:5000
